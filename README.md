# Barrel Run

To play this game, run the 'make' command while in the world06-24trenkt directory to execute the makefile, which will build a game.gb file. Then, bring that file to your emulator of choice to run it. Alternatively, a web-playable version is available on my github page, at https://24trenkt.github.io/

At first, a start screen, such as the one above should appear. Pressing the start key (enter) will begin gameplay, with the main character placed at the bottom of the screen. The user can move the player using the left and right D-pad's left and right keys (arrow keys), and can throw a rock to break barrels using the A button (S). After enough rows of barrels have been passed, the next level will begin! There are a total of four levels, with each getting progressively more challenging, as the barrels spawn closer together, giving the user less time to react. After completing all four levels, the character will run to the middle of the screen and celebrate. Pressing the start key will bring the user back to the start screen, where they can play again!
